# `Aeroway`

A [`Feature`](./feature.md) that is used to tag aerodromes, airfields other ground facilities that support the operation of airplanes and helicopters.

```yml
aeroway:
  value: aerodrome
  match: equals
```

Only the following `value`s are allowed:

* `aerodrome`
* `aircraft_crossing`
* `gate`
* `helipad`
* `heliport`
* `navigationaid`
* `spaceport`
* `terminal`
* `windsock`
